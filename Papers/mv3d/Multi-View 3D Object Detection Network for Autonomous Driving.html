Multi-View 3D Object Detection Network for Autonomous Driving

Multi-View 3D Object Detection Network for Autonomous Driving, Xiaozhi Chen, Huimin Ma, Ji Wan, Bo Li, Tian Xia, International Conference on Computer Vision and Pattern Recognition (CVPR), 2017 (Spotlight)
Github: leeyevi/MV3D_TF
Demo: https://www.youtube.com/watch?v=POqBiiLaslk

Abstract

  • Multi-View 3D networks (MV3D)
  • LIDAR point cloud + RGB images
    • generates 3D candidate boxes efficiently from the bird’s eye view representation of 3D point cloud
    • deep fusion scheme to combine region-wise features from multiple views
  • KITTI 结果（本论文的主要贡献）
    • 3D：outperforms the state-of-the-art by around 25% and 30% $AP_{loc}$、$AP_{3D}$（Average Precision 平均精度） on the tasks of 3D localization and 3D detection（比当前效果最好的还屌）
    • 2D：for 2D detection, our approach obtains 14.9% higher $AP_{2D}$ than the state-of-the-art on the hard data among the LIDAR-based methods

Introduction

  • 本文课题有急切的需求：Modern self-driving cars are commonly equipped with multiple sensors
    • laser scanners：accurate depth information
    • cameras：much more detailed semantic information
    • fusion of LIDAR point cloud and RGB images $\Longrightarrow$ higher performance and safty to self-driving cars
  • 本文课题研究现状演变与本文提出方案的 focus：utilizing both LIDAR and image data $\Longrightarrow$ highly accurate 3D localization and recognition of objects in the road scene
    • LIDAR-based methods vs Image-based method
      • Recent LIDAR-based methods 思路
        • 一种：3D windows in 3D voxel grids to score the point cloud
        • 一种：convolutional networks to the front view point map in a dense box prediction scheme
      • Image-based method 主要思路
        • 首先：generate 3D box proposals
        • 接着：perform region-based recognition using the Fast RCNN pipeline
      • 优势对比，各有千秋
        • Methods based on LIDAR point cloud achieve more accurate 3D locations
        • image-based methods have higher accuracy in terms of 2D box evaluation
    • 已经在尝试的 early or late fusion schemes（传统融合方式）：combine LIDAR and images for 2D detection
      • 存在问题（本文遇到的挑战）$\Longrightarrow$ well-designed model is required to make use of the strength of multiple modalities（多模态数据）
  • 本文提出的解决方案：Multi-View 3D object detection network (MV3D)
    • 利用多模态信息的核心：region-based feature fusion
    • multi-view 3D detection network：3D Proposal Network and a Region-based Fusion Network
      • ① generates 3D object proposals from bird’s eye view map $\Longrightarrow$ generate highly accurate 3D candidate boxes（候选框）
      • ② projecting 3D proposals to the feature maps from mulitple views（多视角） + ROI pooling + deep fusion network $\Longrightarrow$ region-wise fused features
        • superior performance over the early/late fusion scheme（早期或晚期的融合方案）
          • drop-path training：会在训练的迭代过程中随机地删除一些分支
          • auxiliary loss：辅助损失（可能是一种损失函数？？？）
      • ③ fused features jointly $\Longrightarrow$ (1).predict object class (2).do oriented 3D box regression（predict accurate 3D location, size and orientation of objects in 3D space）
  • 本文提出的方法评估：evaluate our approach
    • recent 3D proposal methods 3DOP and Mono3D

  • 3D Proposal Network（3D 提取/检测网络）
    • 避免多检测点（bounding box’s 8 points）冗余的方法：Non-Maximum Suppression(NMS)：非极大抑制
      • 本文使用的：IoU threshold of 0.7 for NMS

        非极大抑制（Non-Maximum Suppression）

        • 在物体检测非极大值抑制应用十分广泛，主要目的是为了消除多余的框，找到最佳的物体检测的位置。
        • Non-Maximum Suppression 就是需要根据 region 属于某个类的 score 矩阵和 region 的坐标信息，从中找到置信度比较高的 bounding box。
          • ①首先，NMS 计算出每一个 bounding box 的面积，然后根据 score 进行排序，把 score 最大的 bounding box 放在队列中。
          • ②接下来，计算其余 bounding box 与当前最大 score 的 box 的 IoU，去除 IoU 大于设定的阈值的 bounding box。
          • ③然后重复上面的①②过程，直至候选 bounding box 为空。
        • 检测 bounding box 的过程中有两个阈值，一个就是 IoU，另一个是在过程之后，从候选的 bounding box 中剔除 score 小于阈值的 bounding box。
        • 需要注意的是：Non-Maximum Suppression一次处理一个类别，如果有 N 个类别，Non-Maximum Suppression 就需要执行 N 次
      • 什么是 IoU(Intersection-over-Union) overlap 呢？

        深度学习（十八）基于R-CNN的物体检测

        • 物体检测需要定位出物体的 bounding box，对于 bounding box 的定位精度，有一个很重要的概念，IoU，IoU 定义了两个 bounding box 的重叠度
        • 矩形框 A、B 的一个重合度 IoU 计算公式为：$ IoU = \frac{A∩B}{A∪B} $，就是矩形框 A、B 的重叠面积占 A、B 并集的面积比例：$ IoU = \frac{S_{A∩B}}{S_A + S_B - S_{A∩B}} $

      神经网络与深度学习-正则化
      $\quad$ 避免过拟合的方法之一是增加训练数据数量。那么，还有没有别的方法能让我们避免过拟合呢？一种可能的方法是减小网络的规模。然而，我们并不情愿减小规模，因为大型网络比小型网络有更大的潜力。

Conclusion and Future Work

  • Conclusion
    • multi-view sensory-fusion model for 3D object detection in the road scene
      • takes advantage of both LIDAR point cloud and images.
      • align different modalities（多模态） by generating 3D proposals and projecting them to multiple views for feature extraction（特征提取）.
      • region-based fusion network is presented to deeply fuse multi-view information and do oriented 3D box regression
    • 在 3D 定位、3D 检测以及 2D 检测都超过 state-of-the-art（目前最先进的方法）
      • outperforms existing LIDAR- based and image-based methods on tasks of 3D localization and 3D detection on KITTI benchmark
      • 2D box results obtained from 3D detections also show competitive performance compared with the state-of-the-art 2D detection methods.
