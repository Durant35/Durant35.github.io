3D SLAM Paper Lists

LOAM

Github: laboshinl/loam_velodyne

LOAM: Lidar Odometry and Mapping in Real-time

We propose a real-time method for odometry and mapping using range measurements from a 2-axis lidar moving in 6-DOF（使用单线激光雷达进行三维重建）

  • V. LIDAR ODOMETRY（里程估算） $\Longrightarrow$ The odometry algorithm estimates motion of the lidar within a sweep（相邻两帧点云数据间的 motion，运动变化）
    • A. Feature Point Extraction（提取每一帧的特征点）
    • B. Finding Feature Point Correspondence（特征点对应（匹配））
    • C. Motion Estimation
      • The lidar motion is modeled with constant angular and linear velocities during a sweep
  • VI. LIDAR MAPPING
    • The mapping algorithm matches and registers $P̄_{k+1}$ in the world coordinates
    • At the end of sweep k+1, the lidar odometry generates a undistorted point cloud, $P̄_{k+1}$, and simultaneously a pose transform, $T^L_{k+1}$, containing the lidar motion during the sweep, between $[t_{k+1}, t_{k+2}]$.

VeloSLAM

Github: victl/VeloSLAM

Dynamic VeloSLAM —— Preliminary（初步） Report on 3D Mapping of Dynamic Environments

主要贡献：We extend the well-known ICP algorithm for HDL-64 laser scan data and build a system for solving the simultaneous localization and mapping problem in urban road scenarios.

  • II. STATE OF THE ART
    • 3D point cloud registration
      • The goal of registration is to find the relative position and orientation of one 3D scan
      • Iterative closest point (ICP) algorithm relies on minimizing the cost function（最小化目标函数）
      • Four closed form solutions are known for minimizing the ICP error function（四种最优解方式）
        • SVD-based solution
        • Solution based on orthonormal matrices
        • Unit quaternion solution
        • Dual quaternion solution
      • Alternatives to ICP
        • Uses a squared distance function
        • Normal distribution transform (NDT)
    • Globally consistent 3D scan matching
      • Approaches like TORO and HOG-Man
      • Gaussian distribution
    • Mapping dynamic environments
      • The localization was done using Monte Carlo
      • Use grid mapping for localization, and then scan subtraction to find dynamic objects
  • V. SLAM IN DYNAMIC ENVIRONMENTS
    • A. Ground and moving objects detection
    • 基于点云特征的动态物体分类这一部分可以继续研读下去[2017.7.5]
